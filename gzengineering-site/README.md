# GZ Engineering Static Website

Αρχική έκδοση static site (HTML/CSS/JS) έτοιμη για GitHub Pages.

## Τοπική προεπισκόπηση
Ανοίξτε το `index.html` ή τρέξτε έναν απλό static server.

## GitHub Pages
Για δοκιμή σε project URL:
`https://username.github.io/gzengineering-site/`

Όλα τα links είναι relative για να λειτουργούν και σε custom domain `https://www.gzengineering.gr/`.
